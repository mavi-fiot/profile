Дипломний проект (2025):

Підсистема криптографічного захисту повідомлень на основі еліптичних кривих Едвардса

  • Розроблено та впроваджено підсистему криптографічного захисту повідомлень, яка здійснює шифрування точки на кривій Едвардса (Ed25519) за геш-значенням повідомлення з використанням SHA3-512 та ElGamal-перетворення, у поєднанні з цифровим підписуванням (EdDSA) і перевіркою достовірності (Ed25519, timestamp).
  • Реалізовано метод точкової передачі повідомлення без розкриття відкритого тексту або його шифрограми.
  • Створено API для інтеграції з іншими модулями обміну повідомленнями, включно з: генерацією ключів, шифруванням повідомлень (точок на кривій), підписом геш-значень, перевіркою підпису, аудитом криптографічних подій і публікацією результатів.
  • Технологічний стек: Python (ecpy, secrets, hashlib, time), FastAPI, SQLite.
